# Proyecto de finalizacion Modulo N° 2 - Sitio E-Commerce
<p>
    <strong>El proyecto consiste</strong> en la creación de un sitio web tipo
    <strong>E-Commerce</strong> para una pastelería, donde se muestran productos
    destacados y un formulario de contacto.
</p>

## Principales secciones de la pagina

<ul>
    <li><strong>Barra de navegación.</strong></li>
    <li><strong>Sección principal (Hero).</strong></li>
    <li><strong>Galería de productos destacados.</strong></li>
    <li><strong>Formulario de contacto.</strong></li>
    <li><strong>Footer.</strong></li>
</ul>
